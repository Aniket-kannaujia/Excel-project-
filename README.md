# 📊 Excel Project - Vrinda Store Data Analysis

## 📌 Project Overview
This project analyzes **Vrinda Store sales data** using Microsoft Excel to uncover business insights related to sales performance, customer demographics, order trends, and channel-wise performance. The project leverages Excel's analytical and visualization capabilities to support data-driven decision-making.

## 🛠️ Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Conditional Formatting
- Excel Formulas

## 📂 Dataset
The dataset contains order-level information, including:
- Order Details
- Customer Information
- Product Categories
- Sales Amount
- Order Status
- Sales Channels
- City & State
- Customer Gender & Age

## 📊 Key Features
- Interactive Sales Dashboard
- Sales Trend Analysis
- Category-wise Performance
- Channel-wise Sales Analysis
- Gender-wise Customer Analysis
- Age Group Analysis
- Order Status Tracking
- State-wise Sales Performance
- Dynamic Filtering using Slicers

## 📈 Business Insights
- Total Sales Performance
- Monthly Sales Trend
- Best Performing Sales Channels
- Top Selling Product Categories
- Customer Demographics
- State-wise Revenue Distribution
- Order Fulfillment Status
- Sales Contribution by Gender and Age Group

## 📋 Excel Skills Demonstrated
- Data Cleaning

- Pivot Charts
- Slicers
- Lookup Functions
- IF Functions
- Conditional Formatting
- Dashboard Design
- Data Visualization
- Business Reporting

## 📁 Project Files

## 🎯 Project Objective
To analyze retail sales data and build an interactive Excel dashboard that helps stakeholders monitor sales performance, understand customer behavior, and identify business opportunities through data visualization and reporting.
